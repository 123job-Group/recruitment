说明:

本图包含了Python职位信息中的一些职位技能

简单分析:

1. 技能关键词分布较广,总的来说分为几个部分: 操作系统-编程语言-数据库端-计算机基础(数据结构/设计模式)-网页服务器端-前端-测试-大数据 等

2. 操作系统方面 linux 占比最高, 如果平时使用 windows ,也需要花点时间学习一下 linux

3. 大部分职位要求求职者除了熟练 Python 之外,还需要了解一门或多门其他语言.

4. 在学习 Python 的同时,数据库学习也不能落下,不管是 sql 还是 nosql ,你得熟悉一门,并且了解另一种

5. 学习 web 开发的同学注意了,django 和 flask 框架为目前招聘主流,同时你仍需要了解 html-css-js 等前端知识
